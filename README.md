# FirebaseChat
Chat application which uses Firestore

## Architecture and used technologies
* Package manager: CocoaPods
* RxSwft
* RxCocoa
* RxDatasources
* Firebase/Auth
* Firebase/Firestore
* FirebaseFirestoreSwift
* MVVM
* Repository pattern


## Features
* Register, Login
* Create new chat, send and receive messages
* Add participants to chats
* Edit profile
